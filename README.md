# sev_oceans

Basic authentication screen

## Installation

Clone the Repository
```bash
  git clone https://github.com/sammulla10/sev_oceans.git
```
Change the directory and install all the packages
```bash
  cd sev_oceans
  flutter pub get
```
Run the app
```bash
  flutter run
```
